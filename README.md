A simple and beginner-friendly User Management (CRUD) application built using Flask + SQLite + SQLAlchemy.
This project helps beginners easily understand how to build backend + frontend applications using Flask.


Features

✔ Add New Users
✔ View User List
✔ Edit User Details
✔ Delete Users
✔ SQLite Database (No setup needed)
✔ Clean UI using HTML + Jinja2
✔ Easy structure for beginners
✔ Fully working CRUD operations


Tech Stack

Python 3

Flask

Flask SQLAlchemy

SQLite

HTML, CSS

Jinja2 Template Engine


CRUD Operations Explained

Create

User enters name & email → stored in database.

Read

All users fetched using SQLAlchemy and displayed on homepage.

Update

Edit fields beside each user → updates database entry.

Delete

Click Delete → removes user record permanently.
